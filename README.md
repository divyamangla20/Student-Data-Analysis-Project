# Student Data Analysis

This is a beginner Python project created using the Pandas library. The project reads student data from a CSV file, performs some basic data analysis, handles missing values, assigns grades, and saves the cleaned data into a new CSV file.

## Features

* Read data from a CSV file
* Display the dataset
* Check the dataset information
* Find missing values
* Fill missing values
* Add a Grade column
* Find the student with the highest marks
* Save the cleaned dataset

## Technologies Used

* Python
* Pandas

## How to Run

1. Clone the repository.

```bash
git clone https://github.com/divyamangla20/Student-Data-Analysis.git
```

2. Open the project folder.

```bash
cd Student-Data-Analysis
```

3. Install Pandas if it is not already installed.

```bash
pip install pandas
```

4. Run the program.

```bash
python analysis.py
```
