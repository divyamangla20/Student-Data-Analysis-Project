# Student Data Analysis

This is a beginner Python project created using **Pandas** and **Matplotlib**. The project reads student data from a CSV file, cleans the data, handles missing values, assigns grades, finds the topper, and creates a bar chart to visualize the grade distribution.

## Features

* Read data from a CSV file
* Display the dataset
* Check missing values
* Fill missing values
* Add a Grade column
* Find the student with the highest marks
* Save the cleaned dataset
* Create a Grade Distribution Bar Chart
* Save the chart in a `charts` folder

## Technologies Used

* Python
* Pandas
* Matplotlib

## How to Run

Clone the repository.

```bash
git clone https://github.com/divyamangla20/Student-Data-Analysis-Project.git
```

Open the project folder.

```bash
cd Student-Data-Analysis-Project
```

Install the required libraries.

```bash
pip install pandas matplotlib
```

Run the program.

```bash
python student_analysis.py
```
